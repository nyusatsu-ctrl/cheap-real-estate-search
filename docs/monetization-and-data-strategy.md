# 月額運用と情報収集の方針

## 推奨プラン案

### 14日間無料トライアル

- 登録から14日間は0円
- 期間中は全機能を利用可能
- 15日目以降は無料利用を停止
- 継続するユーザーだけ月額2,980円の有料プランに申し込む
- 無料登録時にカード情報は求めない

格安不動産探しは、検索、保存、比較、新着確認を実際に使わないと価値が伝わりにくいため、無料期間中に全機能を開放する方が相性がよいです。

### 有料プラン

- 月額2,980円
- 詳細条件検索
- 保存リスト
- 新着、価格変更、成約済みの更新確認
- 掲載元と許諾状態の確認
- 見積もり相談への導線

最初はプランを1種類にして、ユーザーを迷わせない方針にします。プロ向けの高額プランは、利用者の行動と要望が見えてから追加します。

## 見積もり導線

加入者は物件購入前後に追加費用を確認したくなります。月額プラン単体だけでなく、以下の見積もり相談を収益導線にします。

- 名義変更、相続、売買登記
- 解体工事
- リフォーム
- 土木工事、造成、擁壁、排水
- 残置物撤去

初期は相談フォームで受け付け、運営者が提携先へ手配します。将来的には `estimate_requests` テーブルを作り、管理画面で対応状況を管理します。

## 業者手数料モデル

一番手がかからない初期運用は、当サイトが見積もり受付、業者選定、進行管理を行い、工事契約はお客様と業者が直接行う形です。

流れ:

1. お客様が見積もり依頼
2. 運営者が内容を確認
3. 審査済み業者へ依頼
4. 業者が見積額を入力
5. お客様には金額と概要だけを表示
6. 成約後、業者から当サイトへ手数料を支払い

初期手数料率:

- 解体工事: 10%
- 残置物片付け・処分: 10から15%
- リフォーム: 10%
- 土木・造成: 5から10%
- 名義変更・登記: 固定紹介料または個別設定

業者登録は自由に受け付けますが、見積もり参加は審査制にします。確認項目は会社名、対応エリア、対応業務、許可登録、保険、反社チェック同意、手数料率同意です。

## 有料登録前に必ず明記すること

- 無料期間の日数
- 無料期間終了後は自動課金されないこと
- 月額料金
- 解約方法
- 返金条件
- サービス内での現在の契約状態

## 情報収集の進め方

最初から大量サイトを自動取得すると、規約、表記ゆれ、成約済み判定、掲載許諾のリスクが大きくなります。

推奨手順:

1. 取得元を3から5サイトに絞る
2. 取得した物件は `draft` で保存
3. 管理画面で価格、住所、面積、元ページURL、掲載許諾を確認
4. 問題ない物件だけ `published` にする
5. 成約済みや非公開を定期的に `sold` / `draft` に更新する

## 追加しやすい取得元

- 自治体公式の空き家バンク
- 県単位の空き家バンクプラットフォーム
- 0円物件や格安物件に特化したサイト
- 掲載許諾が取りやすい地域不動産会社

## 有料プランの価値にしやすい機能

- 保存条件
- お気に入り物件
- 新着通知
- 価格変更通知
- 成約済み検知
- 情報元別の信頼度表示
- 物件の更新日表示
- 掲載許諾状態の表示
- 複数エリア横断検索

## 注意点

- スクレイピング禁止のサイトは対象にしない
- 元ページURLを必ず表示する
- 写真や本文の転載は許諾なしに行わない
- 初期は自動公開しない
- 物件の正確性は元ページを優先する
